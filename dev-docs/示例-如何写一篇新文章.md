# 示例：如何写一篇对外文章（内部说明）

> 本文档**不参与站点构建**，仅供维护者在本地阅读。对外发布的文章请放在仓库根目录下的 **`docs/articles/`**。

把任意 **对外** Markdown 文章放到 `docs/articles/` 后，在项目根执行：

```bash
npm run article-seo
```

脚本会为各篇补全 `title`、`description`、`keywords`、`date`（面向 Eleventy 的 front matter），然后执行 `npm run build` 或 `npm start`，即可在站内「文章」页看到并访问。

更完整的流程说明见同目录上一级中的 **[docs/articles/README.md](../docs/articles/README.md)**。
