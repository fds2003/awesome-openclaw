# 示例：如何写一篇新文章

这是一篇示例文章。你把任意 Markdown 放到 `docs/articles/` 后，在项目根执行：

```bash
npm run article-seo
```

脚本会为本文补全 `title`、`description`、`keywords`、`date`，然后执行 `npm run build` 或 `npm start` 即可在站内「文章」页看到并访问。
